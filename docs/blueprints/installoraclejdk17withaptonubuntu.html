Step 1 - Install Software Properties Common Package

Connect via ssh:

ssh {ubuntusudouser}@{ubuntunode}

Execute the following script: 

                
apt update
apt install software-properties-common -y

Step 2 - Import the signing key

Execute the following script: 

                
apt-key adv --keyserver keyserver.ubuntu.com --recv-keys EA8CACC073C3DB2A

Step 3 - Add the Java Repository

Execute the following script: 

                
add-apt-repository ppa:linuxuprising/java -y

Step 4 - Install JDK Development Kit 17

Pre-seed the debconf database with answers to silence Oracle License Agreement.

To value key is identified in the oracle-java17-installer.preinst file.

license=oracle-license-v1-3

db_get shared/accepted-$license
if [ "$RET" = "true" ]; then
    echo "$license license has already been accepted" >&2
    exit 0
fi

Execute the following script: 

                
apt update
echo debconf shared/accepted-oracle-license-v1-3 select true | \
  debconf-set-selections
echo debconf shared/accepted-oracle-license-v1-3 seen true | \
  debconf-set-selections
apt install oracle-java17-installer -y

Step 5 - Set the Oracle JDK JAVA_HOME Environment Path

The JAVA_HOME path can be identified by running sudo update-alternatives --config java.

Execute the following script: 

                
echo 'JAVA_HOME="/usr/lib/jvm/java-17-oracle"' >> /etc/environment
source /etc/environment
echo $JAVA_HOME
